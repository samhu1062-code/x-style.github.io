# 📦 外箱尺寸计算器

一个轻量级的在线工具，用于计算**最优外箱尺寸**，帮助包装工程师快速确定纸箱规格，节省包装材料成本。

## 🎯 功能特点

- ✅ **智能优化算法**：自动选择表面积最小的包装方案，减少纸板浪费
- ✅ **多方向排列**：支持小盒任意方向旋转（6种方向），找到最优摆放方式
- ✅ **实时计算**：输入参数后点击计算或按 Enter 键即可获得结果
- ✅ **详细结果输出**：显示外箱尺寸、表面积、体积、排列方式等关键信息
- ✅ **响应式设计**：在电脑、平板、手机上都能正常使用
- ✅ **无需服务器**：纯静态 HTML 页面，可直接在浏览器中运行

## 📋 使用说明

### 输入参数

| 参数 | 说明 | 示例 |
|------|------|------|
| 小盒长度 | 单个小盒的长度 (mm) | 100 |
| 小盒宽度 | 单个小盒的宽度 (mm) | 80 |
| 小盒高度 | 单个小盒的高度 (mm) | 60 |
| 小盒数量 | 需要装入外箱的小盒总数 | 24 |
| 厚度补偿 | 纸箱厚度 + 安全间隙 (mm) | 7.5 |

### 输出结果

- **外箱尺寸**：最优的长 × 宽 × 高 (mm)
- **外箱表面积**：纸板用量 (mm² 和 m²)
- **体积**：外箱容积 (mm³ 和 m³)
- **排列方式**：行 × 列 × 层
- **小盒方向**：计算时采用的小盒摆放方向

### 计算逻辑

算法会遍历所有可能的小盒方向和排列组合，选择**表面积最小**的方案作为最优解。
